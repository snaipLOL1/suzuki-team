## Versioning Scheme

Suzuki Userbot follows [Semantic Versioning](https://semver.org/):
- **MAJOR** version for incompatible API changes
- **MINOR** version for new functionality in a backwards compatible manner
- **PATCH** version for backwards compatible bug fixes

## Based on Hikka Userbot

Suzuki Userbot is based on Hikka Userbot v2.0.4 with the following core components:
- **Hikka-TL**: v2.0.4
- **Hikka-Pyro**: v2.0.103
- Core architecture and module system
- Security implementation
- Inline functionality

## Migration from Hikka

For users migrating from Hikka Userbot, please note:
- Configuration files may need updates
- Module compatibility should be verified
- Backup your data before migration
